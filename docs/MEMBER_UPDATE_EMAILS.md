# Member Update Emails

Use these templates for manual member updates or a future email provider. Do not connect automation yet.

## Weekly Safety Note Email

Subject:
This week's VibeCode Safety Note: [Topic]

Body:

Hi,

This week's Safety Note focuses on [Topic].

What to check:
- [Risk 1]
- [Risk 2]
- [Risk 3]

Open the weekly note in:
`04-Weekly-Safety-Notes/[FOLDER]/`

Run the copy-paste prompt against your latest diff, then use the quick checklist before commit, push, or deploy.

Reminder: keep using the Daily AI Change Review Prompt after AI edits code.

VibeCode Safety helps reduce risk, but it does not guarantee secure, bug-free, or production-ready software.
